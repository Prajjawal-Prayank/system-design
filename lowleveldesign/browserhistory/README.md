# Problem Statement
Design a Browser History System similar to Chrome or Firefox. Handle “Back” and “Forward” navigation. <br>
<br>
Design and implement a class BrowserHistory that supports: <br>
1️⃣ visit(String url) – Visit a new URL and clear forward history. <br>
2️⃣ back() – Go back to the previous page. <br>
3️⃣ forward() – Move forward if available. <br>
4️⃣ getCurrentPage() – Get the current page being viewed. <br>
