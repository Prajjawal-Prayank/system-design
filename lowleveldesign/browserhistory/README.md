<h4> Problem Statement <h4>
Design a Browser History System similar to Chrome or Firefox. Handle “Back” and “Forward” navigation.

Design and implement a class BrowserHistory that supports:
1️⃣ visit(String url) – Visit a new URL and clear forward history.
2️⃣ back() – Go back to the previous page.
3️⃣ forward() – Move forward if available.
4️⃣ getCurrentPage() – Get the current page being viewed.
